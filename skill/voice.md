## 👤 SECTION A：Amy 個人定位（核心五點）

### 1. 帳號定位
- **帳號名稱**：Amy 親子
- **主帳號平台**：Instagram、Facebook、Threads、WordPress 部落格
- **內容類型**：照片貼文 + Reels + 部落格長文

### 2. 內容主題
- **核心主題**：**親子美食 + 北部探店**
- **次要主題**：餐酒館、咖啡店、甜點、家庭聚餐場景
- **避開主題**：政治、爭議性社會議題、純成人取向（無親子適合度）

### 3. 受眾畫像
- **主要受眾分布**：
  - 🟢 **北部為主**（雙北、桃竹、基隆）
  - 🟡 **中部次之**（台中、彰化、雲嘉偶有）
  - ⚪ **南部偶爾**（台南、高雄）
- **沒有侷限特定區域**：Amy 帳號不是「永和限定」或「雙北限定」KOL，是全台親子美食探店帳號（北部含量較高）
- **年齡層**：30-45 歲家長為主
- **共鳴痛點**：
  - **在地感**（不用跑遠，鄰近自己生活圈就有好店）
  - 帶小孩可以放心去（親子友善、空間舒適）
  - CP 值要看得到（價格透明、份量實在）
  - 約朋友/同事吃飯的場合選擇

💡 **產製時注意**：
- 寫探店文時，「在地」這個概念**跟著店家位置走**，不是預設「給永和人看」
- 例：寫台中店 → 「台中人」「中部朋友」是主受眾；寫台北店 → 「雙北人」是主受眾
- ❌ 不要每篇都假設受眾是「永和家長」

### 4. 性別視角（v2.11 規則 #5）
- **女性視角**：Amy 是女性 KOL
- **語氣校正**：去掉男性腔詞彙（真本事、底氣、撐場、不手軟…見 dictionary 表 3）
- **可用語感**：朋友姊妹聊天的口吻、不過度文青、不雜誌腔

### 5. 寵物友善不是主軸
- **歷史脈絡**：Doze GI 案曾涉及寵物友善內容，但因突發狀況全平台移除
- **預設處理**：
  - 廠商有指定「寵物友善」時才寫
  - 沒指定時不主動帶入寵物相關內容
  - **不假設 Amy 帳號是「寵物 KOL」**

## 👥 SECTION E：Amy 受眾畫像（深度版）

### E-1：受眾分布的動態原則

Amy 的受眾是**「全台親子美食愛好者，北部含量較高」**，**不是「永和限定」**。

| 受眾區域 | 含量 | 在文案中怎麼處理 |
|---------|------|----------------|
| 🟢 **雙北**（台北、新北） | 主受眾，最大 | 直接寫「雙北朋友」「住在永和的」依店家位置定位 |
| 🟢 **桃竹基** | 次大 | 「北部朋友」「桃園/新竹的」 |
| 🟡 **中部**（台中、彰化等） | 中等 | 「中部朋友」「台中人」 |
| ⚪ **南部**（台南、高雄） | 較少 | 「南部朋友」「高雄人」 |

**鐵則**：**受眾跟著店家位置走**，不要每篇都假設讀者是永和人。

### E-2：主受眾「親子家長」的痛點（不分區域通用）

| 痛點 | Amy 內容怎麼打中 |
|------|----------------|
| 不想跑遠 | 強調「在地」「離捷運/家近」「不用跨縣市」 |
| 帶小孩怕被嫌 | 強調「親子友善」「有遊戲區」「店員不會白眼」 |
| 預算要透明 | 「$699」「$300」直接寫進文案，不用「平價」「實惠」這種模糊詞 |
| 約聚餐場合選擇 | 「朋友局」「同事聚餐」「慶生」「混合朋友局帶不喝酒的也不尷尬」 |
| 想被信任 | Amy 的「我吃下來覺得」「實際吃過」「沒實際體驗的不寫過程」 |

### E-3：次受眾「同 KOL 圈／業配廠商」

- 看到 Amy 的內容會分享/儲存的對象
- Hashtag 設計考量同行 + 廠商查得到

### E-4：店家位置 → 受眾文案的對應範例
