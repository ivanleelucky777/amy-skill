### 🔍 深度詞解釋（最常踩雷的，附「為什麼禁」）

| 詞 | 為什麼禁 |
|---|---|
| **銷魂** | 太誇大 + 帶性暗示 + 不是日常用語，廣告腔代表 |
| **入口即化** | 99% 食物文案的萬用懶詞，失去個人觀點，AI 痕跡明顯 |
| **欲罷不能** | 成語包裝的廣告腔，真人朋友不會這樣講 |
| **絕品** | 涉嫌違反《公平交易法》「不得使用第一、最、唯一、絕」精神 |
| **層次豐富** | 美食文案的萬用懶詞，沒實質描寫，純評論員腔 |
| **彷彿置身** | 雜誌寫作公式，KOL 用了就出戲 |
| **療癒系/儀式感** | 文青行銷腔，過度詮釋食物 |
| **靈魂** | 「靈魂醬料」「靈魂配菜」這類，無實質意義的浮誇詞 |
| **爆漿** | 過度感官化、AI 痕跡明顯（這次討論已經移除過） |

## 表 2｜AI 寫作符號禁用表

| 符號/句型 | 範例 | 替換 |
|---|---|---|
| `——` 雙破折號 | 「松露燉飯——主角中的主角」 | 用句號或逗號 |
| `—` 單破折號 | 「禮拜五是最划算的一天—三項一起跑」 | 用逗號 |
| 「不是 X，而是 Y」 | 「不是普通燉飯，而是松露燉飯」 | 直接寫「松露燉飯，米心保留硬度」 |
| 「讓 X 變得更 Y」 | 「松露讓燉飯變得更香」 | 直接寫「松露味從第一口跑出來、越吃越濃」 |
| 「在 X 的同時，也 Y」 | 「在保留嚼勁的同時，也兼顧香氣」 | 直接寫「咬下去有嚼勁、香氣也夠」 |
| 連續引號 `「『 』『 』」` | 雙層引號嵌套 | 重寫，拆成兩句 |
| 「X，更是 Y」對仗 | 「不只是燉飯，更是一場饗宴」 | 拆成白話兩句 |
| 同段結尾用同個句型 | 三段結尾都是「就是這麼簡單」 | 改變句型 |

## 表 7｜廣告法風險詞（《公平交易法》/《食品安全衛生管理法》）

**這些詞可能觸法或進入廣告法灰色地帶，⚠️ 強烈建議不用**：

| 風險詞 | 風險類型 | 替換 |
|---|---|---|
| 最 X、第一、唯一、頂級、巔峰 | 「最高級用語」明文禁止 | 改限定範圍：「永和最 X」「店家自慢的」 |
| 保證、絕對、肯定、絕無 | 絕對性用語 | 「我吃下來覺得」「實際吃到的是」 |
| 療效、功效、神奇、療癒（健康宣稱） | 食品宣稱療效違法 | 純口感描寫 |
| 美容、養顏、補身、養生（食品宣稱） | 涉醫療廣告 | 純口感描寫 |
| 排毒、減肥、瘦身（食品宣稱） | 涉醫療廣告 | 不用 |
| 神級、極品、絕品 | 接近最高級用語 | 「店家拿手的」「值得試」 |
| 真材實料、用料新鮮（無證明時） | 不實宣稱風險 | 「我吃下來覺得新鮮」（個人感受） |

### 例外：限定範圍的「最」可用
- ❌「全台最狂」、「台北最大碗」
- ✅「**永和**最狂」、「**仁愛路上**最大碗」、「**這條街**最熱鬧」

## 表 8｜腦補高風險詞（v2.8 + Doze GI 學習）

**Claude 看到下列類型的內容時，先停下來確認，不要腦補**：

| 高風險區 | 為什麼危險 | 處理方式 |
|---|---|---|
| **店家品類**（義式/串燒/居酒屋/餐酒館） | 不同品類完全寫不一樣 | 用廠商原始素材的描述，不確定時問 Ivan |
| **地理位置**（縣市/區） | SEO 錯誤 + 廣告不實 | 嚴格照素材，永和 ≠ 台北 |
| **菜名/調味/做法** | 菜單外的詳細描述 | 只用廠商提供的菜單名，不自編 |
| **食材成分**（益生菌菌株、添加物） | 涉醫療廣告法規 | 以 brief 提供為準，沒寫的不加 |
| **老闆/主廚個人名字** | 可能不是真名或有隱私顧慮 | 用廠商素材原文，不確定就稱「店家」「主廚」 |
| **店家歷史/開店動機** | 容易腦補出感人故事 | 純素材轉述，不寫小說 |
| **份量數字**（吃到飽幾顆、營業幾年） | 跟廠商活動名稱不符會穿幫 | 不寫具體數字，用「給很多」「夠分」帶過 |
| **體驗過程**（吃幾輪、加什麼醬） | KOL 沒實際體驗就寫過程 = 造假 | 看 `principles.md` 體驗真實性分流 |
| **店家分層調酒/特殊技法** | 容易腦補成「漸層」「分子料理」 | 廠商有明說才寫，沒明說就用一般描述 |

## 表 9｜AI 痕跡偵測 vs KOL 標配（v2.7 重要區分）

### ❌ AI 痕跡（被 IG 演算法判定為 AI 生成內容會降權）

```
- 句型完美工整（每段字數差不多、每句結構一致）
- 三段並列句（「不只 X、不只 Y、更是 Z」）
- 結尾每段都是同個句型（「就是這麼簡單」「值得一試」）
- 過度排比（「鮮的、甜的、香的、嫩的」連用四個以上）
- 無口語贅詞、語助詞（完全沒有「啊」「欸」「就」「對」）
- 結構太對稱（首尾呼應太刻意）
- 所有形容詞都精準到不像真人講話
```

### ✅ KOL 標配（鼓勵使用、不要誤判為 AI 痕跡）

```
- emoji：📍 🍝 🍹 🌈 🦪 ⚡️ 等
- 條列符號：✅ ❌ 📅 🕐 💬 📌
- 短句斷行（IG/FB 標配排版）
- 口語贅詞：「啊」「欸」「就」「對」「真的」「有點」「蠻」
- 中英混用：「真的很 Q」「CP 值高」「這個 vibe」
- 數字 emoji 強調價格：「$799」「$300」
- 跳行加 emoji：「📍 永和仁愛路 213 號」
- 段落間穿插一句感嘆：「（真的，沒騙你）」
```

### 判斷尺：
- AI 痕跡 = 文字「太工整、太完美、沒口語感」
- KOL 標配 = 視覺輔助、口語化排版、emoji 點綴
